About me

As a user interface designer and front end developer, I believe website creation is an art. Websites have become one of the main ways customers interact with businesses. Every time I make a website, my goal is to create an online space where people can connect with the companies they care about. Every website provides a unique challenge, and I’m always up to the task of creating a design that is easy to use and visually beautiful. I have a passion for the language of computer code, and I’m constantly improving my skills in HTML, CSS, and jQuery.

Web Design

Crafting the user’s experience is the most important aspect of web design. Websites exist to deliver information, and in my work as a user interface designer, it’s my job to ensure that task is done well. I create simple and efficient web experiences so that your client can find what they need, and leave your site with a pleasant impression of your business.

Code

Clean code is key to creating a well-functioning website. As a front end developer, I lay the foundation for websites, create the visual style, and ensure cross-browser and cross-device functionality – all in coding languages. This is the building blocks of your website, and my goal is for the coding to be so well done, you won’t have to worry about it all.
